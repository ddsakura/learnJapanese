# 題庫匯入/正規化規則

## 支援輸入格式
- JSON 陣列
- 元素可為：
  - 字串（只含 dict）
  - 物件（可含 dict + 其他欄位）

## 正規化流程（高層）
1. 若元素為字串：
   - 動詞：推斷 group → 推導所有形
   - 形容詞：推斷 group（含 〜だ）→ 推導所有形
2. 若元素為物件：
   - 若完整且欄位合法，直接使用
   - 否則以 dict + group 推導，再用提供欄位覆寫

## 可覆寫欄位
- 動詞：nai, ta, nakatta, te, potential, zh
- 形容詞：nai, ta, nakatta, te, zh

## 錯誤情境（需回傳錯誤）
- JSON 不是陣列
- 空白項目
- 物件缺少 dict
- 無法推導（例如未知動詞）
