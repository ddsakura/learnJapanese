# 變化規則摘要

## 動詞

### 不規則
- する系：
  - ない形：語幹 + しない
  - た形：語幹 + した
  - なかった形：語幹 + しなかった
  - て形：語幹 + して
  - 可能形：語幹 + できる
- くる系：
  - ない形：語幹 + こない
  - た形：語幹 + きた
  - なかった形：語幹 + こなかった
  - て形：語幹 + きて
  - 可能形：語幹 + こられる

### 一段（ichidan）
- 語幹 + ない / た / なかった / て / られる

### 五段（godan）
- う/つ/る：
  - ない形：語幹 + わない / たない / らない
  - た形：語幹 + った
  - て形：語幹 + って
  - 可能形：語幹 + える / てる / れる
- む/ぶ/ぬ：
  - ない形：語幹 + まない / ばない / なない
  - た形：語幹 + んだ
  - て形：語幹 + んで
  - 可能形：語幹 + める / べる / ねる
- く（行く例外）
  - 行く：た形=行った / て形=行って
  - 其他：た形=語幹+いた / て形=語幹+いて
  - ない形：語幹 + かない
  - 可能形：語幹 + ける
- ぐ：
  - ない形：語幹 + がない
  - た形：語幹 + いだ
  - て形：語幹 + いで
  - 可能形：語幹 + げる
- す：
  - ない形：語幹 + さない
  - た形：語幹 + した
  - て形：語幹 + して
  - 可能形：語幹 + せる

## 形容詞

### い形容詞
- 語幹 + くない / かった / くなかった / くて
- いい（例外）：
  - ない形：よくない
  - た形：よかった
  - なかった形：よくなかった
  - て形：よくて

### な形容詞
- 語幹 + じゃない / だった / じゃなかった / で
- 若原形為「〜だ」，先去掉「だ」再變化
