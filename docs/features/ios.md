# iOS App 功能說明

## App 概覽

JLPT N4 日語活用練習 iOS 應用，以 SwiftUI 開發，提供動詞與形容詞的普通形活用練習（ない形／た形／なかった形／て形／可能形），以及自他動詞配對與判斷練習。內建 SRS 間隔複習機制，支援文字輸入與 AI 四選一兩種作答模式，AI 功能由 Apple Intelligence 驅動（本機執行，無需網路）。

---

## 畫面清單

### 主練習畫面

- **對應檔案**：`apps/ios/Sources/ContentView.swift`、`apps/ios/Sources/AppState.swift`
- **功能說明**：應用的核心畫面，以 ScrollView 呈現題目卡片、答題區、批改結果、統計、AI 翻譯例句，所有練習功能都在此單一畫面完成。
- **主要 UI 元素**：
  - 頂部導覽列：標題「LearnJapanese」＋右上角「題庫管理」按鈕
  - 學習設定摘要列：顯示目前設定（如「活用練習・自他動詞・找配對」），點擊開啟設定 Sheet
  - 題目卡片（QuestionCardView）：辭書形大字＋目標活用形標示
  - 朗讀按鈕：使用 AVSpeechSynthesizer 播放日文語音
  - 答題區：文字輸入框（Input 模式）或選項按鈕組（Choice 模式）
  - 批改／略過按鈕
  - 批改結果卡片：對錯標示、使用者答案與正確答案對照
  - 統計卡片：今日答題數、連續答對、今日答錯
  - AI 翻譯與例句區塊（Apple Intelligence 生成）
  - 下一題按鈕
- **使用者操作流程**：
  1. 閱讀學習設定摘要列，確認目前設定
  2. 查看題目卡片（辭書形 → 目標活用形）
  3. 點擊朗讀按鈕聆聽發音（選填）
  4. 在輸入框輸入答案，或點選四個選項之一
  5. 點「批改」（輸入模式）或點選選項後自動批改
  6. 查看對錯結果、AI 生成的中文翻譯與日文例句
  7. 點「下一題」進入下一題
- **導航關係**：
  - 從：應用啟動後直接進入
  - 到：點「題庫管理」開啟題庫管理 Sheet；點學習設定摘要列開啟設定 Sheet
- **特殊狀態**：
  - Empty（正常模式）：題庫無可用題目時顯示「目前題庫沒有可用題目」
  - Empty（複習模式）：今日無答錯時顯示「目前沒有可複習錯題」
  - Loading（AI）：Apple Intelligence 生成中顯示 ProgressView ＋「翻譯與例句產生中…」
  - Error（AI）：模型不可用時顯示錯誤說明文字（如「AI 模型不可用，請確認 Apple Intelligence 已啟用」）

### 題目卡片（QuestionCardView）

- **對應檔案**：`apps/ios/Sources/QuestionCardView.swift`
- **功能說明**：以卡片樣式呈現單一題目，顯示辭書形與目標活用形，作為使用者答題的主要視覺參考。
- **主要 UI 元素**：
  - 辭書形：大號字（36pt 粗體）
  - 箭頭符號「→」
  - 目標活用形標籤：綠色字（20pt），如「た形」、「可能形」
  - 圓角卡片背景＋陰影
- **使用者操作流程**：
  1. 閱讀辭書形
  2. 確認右側的目標活用形
  3. 在卡片下方的答題區作答
- **導航關係**：嵌入於主練習畫面，無獨立導航

---

### 學習設定 Sheet（SettingsView）

- **對應檔案**：`apps/ios/Sources/ContentView.swift`（SettingsView struct）
- **功能說明**：以 Modal Sheet 呈現所有學習偏好設定，套用後重新生成題目。設定包含學習主題、練習對象、作答方式、題型、詞彙範圍。
- **主要 UI 元素**：
  - 學習主題 Picker（Segmented）：「活用練習」／「自他動詞」
  - 學習對象 Picker（活用練習時）：「動詞」／「形容詞」
  - 作答方式 Picker（Segmented）：「文字輸入」／「四選一」
  - 題型清單（List，含 checkmark）：ない形／た形／なかった形／て形／可能形／混合
  - 動詞種類清單（動詞時）：全部／五段／二段／不規則
  - 形容詞種類清單（形容詞時）：全部／い形／な形
  - 自他動詞題型清單（自他動詞時）：找配對／判斷自他
  - 取消按鈕（左上）、套用按鈕（右上）
- **使用者操作流程**：
  1. 點擊主畫面的設定摘要列，Sheet 從下方滑出
  2. 選擇學習主題（活用練習或自他動詞）
  3. 依主題調整對應選項
  4. 點「套用」儲存設定並重新產生題目
  5. 或點「取消」不儲存直接關閉
- **導航關係**：
  - 從：主練習畫面點擊設定摘要列呼出
  - 到：點「套用」或「取消」後回到主練習畫面
- **特殊狀態**：無

### 自他動詞練習區（主練習畫面內）

- **對應檔案**：`apps/ios/Sources/ContentView.swift`（TransitivityCardView struct）
- **功能說明**：當學習主題切換為「自他動詞」時，主畫面改為顯示自他動詞題目卡片，支援「找配對」與「判斷自他」兩種題型。
- **主要 UI 元素**：
  - 題目詞彙：大號字（40pt）
  - 讀音（假名，選填）
  - 中文翻譯（選填）
  - 題目提示：綠色粗體（如「対になる動詞は何ですか？」或「自動詞ですか？他動詞ですか？」）
  - 批改結果：✅/❌ 標示、正確答案、自他配對顯示（自動詞 ↔ 他動詞）
  - 選項按鈕組（Choice 模式）或文字輸入框（Input 模式）
  - 略過、下一題按鈕
- **使用者操作流程**：
  1. 在設定 Sheet 將主題切換為「自他動詞」並套用
  2. 閱讀題目詞彙與題型提示
  3. 輸入答案或點選選項
  4. 查看批改結果與配對說明
  5. 點「下一題」繼續
- **導航關係**：切換學習主題後直接在主練習畫面呈現，無獨立路由
- **特殊狀態**：
  - Empty：自他動詞題庫為空時顯示「題庫沒有自他動詞資料」

### 題庫管理 Sheet

- **對應檔案**：`apps/ios/Sources/ContentView.swift`（showBankSheet Sheet 區段）
- **功能說明**：以 Modal Sheet 提供題庫的匯入、匯出、快速新增與重置功能，JSON 格式，含 TextEditor 可直接編輯。
- **主要 UI 元素**：
  - 目前題庫數量顯示
  - JSON 文字編輯區（TextEditor）：可貼上或直接編輯題庫 JSON
  - 匯出題庫按鈕：將目前題庫輸出至編輯區
  - 匯入題庫按鈕：解析編輯區 JSON 並合併
  - 重置題庫按鈕：恢復預設題庫並清除 SRS 進度
  - 快速輸入欄位（TextField）：輸入辭書形，以空白或逗號分隔
  - 直接匯入按鈕：快速新增詞彙
  - 操作訊息文字：顯示成功或失敗訊息
  - 完成按鈕（Toolbar 右上）
- **使用者操作流程**：
  1. 點擊導覽列右上角「題庫管理」開啟 Sheet
  2. 選擇操作：
     - 快速新增：在快速輸入欄位輸入辭書形 → 點「直接匯入」
     - JSON 匯入：貼上 JSON 至編輯區 → 點「匯入題庫」
     - 備份：點「匯出題庫」取得 JSON，複製備用
     - 重置：點「重置題庫」恢復預設
  3. 查看操作結果訊息
  4. 點「完成」關閉 Sheet
- **導航關係**：
  - 從：主練習畫面點「題庫管理」呼出
  - 到：點「完成」回到主練習畫面
- **特殊狀態**：
  - Loading：匯入處理中所有按鈕禁用（`isImporting = true`）
  - Error：JSON 解析失敗時顯示「匯入失敗：JSON 解析錯誤。」
  - Success：匯入成功顯示「匯入成功，已合併題庫。」

### AI 翻譯與例句區塊（主練習畫面內）

- **對應檔案**：`apps/ios/Sources/ContentView.swift`、`apps/ios/Sources/AppleIntelligenceService.swift`
- **功能說明**：批改後自動呼叫 Apple Intelligence 生成中文翻譯與日文例句，展示在批改結果下方。使用者亦可手動重新生成。
- **主要 UI 元素**：
  - 進度指示器＋「翻譯與例句產生中…」（生成中）
  - 中文翻譯卡片
  - 例句卡片：日文例句、假名讀音、中文翻譯、文法解說
  - 朗讀例句按鈕
  - 重新產生翻譯／例句按鈕
- **使用者操作流程**：
  1. 批改後本區自動出現並開始生成
  2. 等待生成完成後查看翻譯與例句
  3. 可點「朗讀例句」聆聽發音
  4. 可點「重新產生翻譯／例句」獲取不同例句
- **導航關係**：嵌入於主練習畫面批改結果下方，無獨立路由
- **特殊狀態**：
  - Loading：顯示 ProgressView ＋「翻譯與例句產生中…」
  - Error（Guardrail）：顯示「AI 回覆被安全機制攔截，請重試」
  - Error（不可用）：顯示「AI 模型不可用，請確認 Apple Intelligence 已啟用」

---

### 統計與複習區（主練習畫面內）

- **對應檔案**：`apps/ios/Sources/ContentView.swift`
- **功能說明**：顯示本次練習統計，並提供「複習今日答錯」的入口按鈕。
- **主要 UI 元素**：
  - 統計卡片列：今日答題數、連續答對、今日答錯（三欄並排）
  - 複習今日答錯按鈕：今日有答錯時啟用
  - 回到正常題庫按鈕：複習模式時顯示
- **使用者操作流程**：
  1. 練習過程中隨時查看統計數字
  2. 有答錯時點「複習今日答錯」進入複習模式（只從今日答錯的題目出題）
  3. 複習完畢或想退出時點「回到正常題庫」
- **導航關係**：固定顯示於主練習畫面中段，無獨立路由
- **特殊狀態**：
  - 無答錯：「複習今日答錯」按鈕呈灰色禁用狀態
