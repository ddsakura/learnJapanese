# Web App 功能說明

## App 概覽

JLPT N4 日語活用練習 Web 應用，以單頁應用（SPA）形式呈現，提供動詞與形容詞的普通形活用練習（ない形／た形／なかった形／て形／可能形），以及自他動詞配對與判斷練習。內建簡易 SRS 間隔複習機制，支援文字輸入與 AI 四選一兩種作答模式，並可自訂題庫。

---

## 畫面清單

### 主練習畫面（活用練習・文字輸入模式）

- **對應檔案**：`apps/web/src/App.tsx`、`apps/web/src/components/QuestionCard.tsx`
- **功能說明**：顯示一張動詞或形容詞卡片，要求使用者輸入指定活用形的答案，批改後給予對錯回饋並顯示詳細說明。
- **主要 UI 元素**：
  - 頂部控制欄（HeaderControls）：主題、類型、題型、範圍、作答方式五個下拉選單
  - 題目區：辭書形大字顯示、目標活用形標示（如「た形」）
  - 發音按鈕：點擊播放日文語音（依賴瀏覽器 Web Speech API）
  - 查字典連結：開啟外部字典查詢辭書形
  - 輸入框：輸入日文答案（Enter 送出）
  - 批改按鈕：提交答案進行批改
  - 略過按鈕：跳過本題不計分
  - 下一題按鈕：批改後才啟用，前往下一題
- **使用者操作流程**：
  1. 從頂部下拉選單設定練習範圍（主題／類型／題型／範圍）
  2. 閱讀題目（辭書形 → 目標形）
  3. 在輸入框輸入答案，按 Enter 或點「批改」
  4. 查看對錯結果、正確答案、中文翻譯、例句
  5. 點「下一題」繼續
- **導航關係**：單頁無路由，切換頂部下拉選單即切換練習模式
- **特殊狀態**：
  - Empty：題庫無符合範圍的題目時顯示「目前題庫沒有可用題目」
  - Loading（翻譯中）：批改後 Ollama 正在取得中文翻譯時顯示「（翻譯中…）」
  - Loading（例句中）：Ollama 正在生成例句時顯示「例句產生中…」
  - Error（例句）：Ollama 無法連線時顯示「例句產生失敗，請確認 Ollama 已啟動…」

### 主練習畫面（活用練習・四選一模式）

- **對應檔案**：`apps/web/src/App.tsx`、`apps/web/src/components/QuestionCard.tsx`
- **功能說明**：與文字輸入模式相同的練習流程，但答案改為四個選項，選項由 Ollama 動態生成（一個正確 + 三個干擾項）。
- **主要 UI 元素**：
  - 選項列表：四個可點擊按鈕，點選後立即批改
  - 正確選項：點選後顯示綠色樣式
  - 錯誤選項：點選後顯示紅色樣式
  - 重新產生選項按鈕：強制重新向 Ollama 請求新的選項組合
  - 略過、下一題按鈕
- **使用者操作流程**：
  1. 切換頂部「作答方式」為「四選一」
  2. 等待 Ollama 生成四個選項（顯示「選項產生中…」）
  3. 點選其中一個選項，立即批改
  4. 查看對錯結果與詳細說明
  5. 點「下一題」繼續
- **導航關係**：與文字輸入模式共用同一畫面，切換「作答方式」下拉選單即切換
- **特殊狀態**：
  - Loading：等待 Ollama 生成選項時顯示「選項產生中…」
  - Error：Ollama 無法連線時顯示「選項產生失敗，請確認 Ollama 已啟動…」並顯示「重新產生選項」按鈕

### 批改結果展示區（答題後）

- **對應檔案**：`apps/web/src/components/QuestionCard.tsx`
- **功能說明**：使用者提交答案後，在題目卡下方展開顯示詳細的批改結果與延伸學習資料。
- **主要 UI 元素**：
  - 對錯標示：✅ 正確 或 ❌ 錯誤／略過
  - 題型說明：「題型：た形」
  - 使用者答案與正確答案對照
  - 查字典連結（正確答案形）
  - 中文翻譯（Ollama 即時取得，或卡片內建）
  - 例句區：日文例句、假名讀音、中文翻譯、文法解說
  - 例句朗讀按鈕
  - 全形展示表：列出該詞所有活用形（辭書形／ない形／た形／なかった形／て形／可能形）
- **使用者操作流程**：
  1. 批改後自動展開本區
  2. 瀏覽正確答案、翻譯、例句
  3. 可點擊「查字典」查閱外部資料
  4. 可點擊「朗讀例句」聆聽發音
  5. 點「下一題」關閉本區並進入下一題
- **導航關係**：為 QuestionCard 的展開區塊，無獨立路由

### 自他動詞練習畫面

- **對應檔案**：`apps/web/src/App.tsx`（`renderTransitivitySection` 函式）
- **功能說明**：練習自動詞與他動詞的配對關係，支援「找配對」（給出一個詞，填入對應的自／他動詞）與「判斷自他」（判斷給出的詞是自動詞或他動詞）兩種題型。
- **主要 UI 元素**：
  - 題目區：顯示動詞（如「閉める」）與問題（「対になる動詞は何ですか？」或「自動詞ですか？他動詞ですか？」）
  - 文字輸入框 或 四個選項按鈕（依作答方式而定）
  - 判斷自他模式：自動詞／他動詞 兩個選項按鈕
  - 略過、下一題按鈕
  - 批改結果：顯示自動詞↔他動詞配對、正確答案、讀音
- **使用者操作流程**：
  1. 切換頂部「主題」為「自他動詞」
  2. 選擇題型（找配對 或 判斷自他）與作答方式
  3. 閱讀題目，輸入或選擇答案
  4. 查看批改結果
  5. 點「下一題」繼續
- **導航關係**：切換頂部「主題」下拉選單至「自他動詞」即進入本畫面
- **特殊狀態**：
  - Empty：自他動詞題庫為空時顯示「題庫沒有自他動詞資料」

### 統計面板（StatsPanel）

- **對應檔案**：`apps/web/src/components/StatsPanel.tsx`
- **功能說明**：顯示本次練習的即時統計數據，並提供進入「複習今日答錯」模式的入口。
- **主要 UI 元素**：
  - 今日答題數：本日累計答題次數
  - 連續答對數：目前連續正確題數（答錯歸零）
  - 待複習數：SRS 逾期待複習的卡片數
  - 今日答錯數：本日答錯次數
  - 目前範圍：目前篩選範圍標籤（如「全部」）
  - 複習今日答錯按鈕：今日有答錯時啟用
  - 回到正常題庫按鈕：在複習模式時顯示
- **使用者操作流程**：
  1. 練習中隨時查看面板數據
  2. 有答錯時點「複習今日答錯」進入複習模式
  3. 複習完畢後點「回到正常題庫」
- **導航關係**：固定顯示於主畫面下方，無獨立路由
- **特殊狀態**：
  - 今日無答錯：「複習今日答錯」按鈕灰色禁用

### 變化規則參考面板（RulesPanel）

- **對應檔案**：`apps/web/src/components/RulesPanel.tsx`
- **功能說明**：以可展開的折疊區塊顯示活用變化規則，供使用者在練習時快速查閱。
- **主要 UI 元素**：
  - 折疊區塊標題：「た形・て形・可能形 變形規則」等
  - 動詞規則：五段動詞（各行變化）、二段動詞、特殊動詞（する/くる）、特例（行く）
  - 形容詞規則：い形容詞、な形容詞、特殊（いい→よい）
- **使用者操作流程**：
  1. 點擊折疊標題展開規則表
  2. 查閱規則後再次點擊收合
- **導航關係**：固定顯示於主畫面下方，無獨立路由

### 題庫管理面板（BankPanel）

- **對應檔案**：`apps/web/src/components/BankPanel.tsx`
- **功能說明**：讓使用者匯入、匯出、重置自訂題庫，並可清除學習進度紀錄（SRS 狀態與統計），以可展開的折疊區塊呈現。
- **主要 UI 元素**：
  - 匯入說明文字與 JSON 格式範例
  - 目前題庫總數顯示（如「目前題庫共有 124 個單字」）
  - JSON 文字框：貼上或檢視題庫 JSON
  - 快速輸入框：直接輸入辭書形（空白或逗號分隔），按「直接匯入」快速新增
  - 匯出題庫按鈕：將現有題庫輸出至文字框
  - 匯出全部題庫按鈕：下載含所有類型的 `bank-export.json`
  - 匯入題庫按鈕：解析文字框 JSON 並合併至現有題庫
  - 重置題庫按鈕：恢復預設題庫並清除進度
  - 清空學習紀錄按鈕：僅清除 SRS 與統計，保留題庫
  - 操作訊息區：顯示匯入成功／失敗等訊息
- **使用者操作流程**：
  1. 點擊「題庫管理」折疊區塊展開
  2. 選擇操作：
     - 快速新增：在快速輸入框輸入辭書形 → 點「直接匯入動詞」
     - JSON 匯入：貼上 JSON 至文字框 → 點「匯入題庫」
     - 備份：點「匯出全部題庫」下載 JSON 檔案
     - 重置：點「重置題庫」恢復預設
  3. 查看操作結果訊息
- **導航關係**：固定顯示於主畫面最下方，無獨立路由
- **特殊狀態**：
  - Loading：匯入處理中時所有按鈕禁用，`isImporting=true`
  - Error：JSON 解析失敗時顯示「匯入失敗：JSON 解析錯誤。」
  - Success：匯入成功顯示「匯入成功，已合併題庫。」
